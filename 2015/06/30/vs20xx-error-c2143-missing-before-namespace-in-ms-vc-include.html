VS20XX 错误 Error C2143 (missing ; before namespace) in MS VC include

30 Jun, 2015 · by admin · Read in about 1 min · (14 Words)
visualstudio windows 开发 编译错误

错误信息：

VC\include\yvals.h(535): error C2143: syntax error : missing ‘;’ before ‘namespace’

可能的原因：

你include文件头文件中有文件遗漏了分号。请仔细检查你的源代码！
