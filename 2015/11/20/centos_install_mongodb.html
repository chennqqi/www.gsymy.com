CentOS安装mongodb

20 Nov, 2015 · by admin · Read in about 1 min · (20 Words)

yum源中添加mongodb下载地址

具体步骤：

Cd /etc/yum.repo.d/

创建 mongodb.repo

添加内容

[mongodb]
name=MongoDB Repository
baseurl=http://downloads-distro.mongodb.org/repo/redhat/os/x86_64/
gpgcheck=0
enabled=1


执行sudo yum install -y mongodb-org
